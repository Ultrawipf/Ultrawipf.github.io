◆ command()

ParseStatus FFBoardMain::command ( ParsedCommand *  cmd,
std::string *  reply 
)
virtual

Implement this function. MUST return not found when no valid command was found or if a help command or similar was parsed. When it returns OK or FAIL parsing is normally stopped after this class and the command is not sent to others. A command can not start with "!" or contain ">" anywhere in the reply or command name.

Reimplemented from CommandHandler.
