17 /* Add your addresses here. 0xffff is invalid as it marks an erased field.
18 Anything below 0x00ff is reserved for system variables.
19 
20 Use ranges that are clear to distinguish between configurations. Address ranges can have gaps.
21 Label the names clearly.
22 Example: 0x0100 - 0x01ff for one class and 0x0200-0x02ff for another class would be reasonable even if they each need only 3 variables
23 
24 
25 Important: Add your variable to the VirtAddVarTab[NB_OF_VAR] array in eeprom_addresses.c!
26 
27 Tip to check if a cell is intialized:
28 uint16_t EE_ReadVariable(uint16_t VirtAddress, uint16_t* Data) will return 1 if the address is not found or 0 if it was found.
29 */
