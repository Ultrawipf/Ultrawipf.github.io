Detailed Description

Implements an interface for parsed command handlers.

Adds itself to a global vector of handlers that can be called from the main class when a command gets parsed.

For example motor drivers and button sources can implement this to easily get serial commands

)
virtual

Implement this function. MUST return not found when no valid command was found or if a help command or similar was parsed. When it returns OK or FAIL parsing is normally stopped after this class and the command is not sent to others. A command can not start with "!" or contain ">" anywhere in the reply or command name.

Reimplemented in ODriveCAN, Axis, TMCDebugBridge, TMC4671, SPI_Buttons, ShifterAnalog, MotorPWM, MidiMain, LocalButtons, LocalAnalog, FFBWheel, FFBoardMain, EncoderLocal, CustomMain, CanBridge, NormalizedAxis, and EffectsCalculator.
